# Game Desktop Application

This is a desktop game built using [Electron.js](https://www.electronjs.org/). Enjoy a seamless gaming experience on your PC!

## Features

- Cross-platform desktop support (Windows, macOS, Linux)
- Easy installation
- Engaging gameplay

## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [Git](https://git-scm.com/)

### Steps

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/game-desktop-app.git
    cd game-desktop-app
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Run the game:**
    ```bash
    npm start
    ```

## Packaging the App

To create an installable package for your OS:

```bash
npm run package
```

Find the installer in the `dist/` folder.
